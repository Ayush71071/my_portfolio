## рҹҺЁ Design System

The entire color palette is driven by CSS custom properties:

```css
:root {
  --neon-blue:   #00f0ff;   /* Cyan вҖ” primary accent     */
  --neon-pink:   #ff007f;   /* Magenta вҖ” secondary accent */
  --neon-green:  #00ff66;   /* Lime вҖ” success / code      */
  --neon-yellow: #ffea00;   /* Yellow вҖ” highlights        */
}
```

### Typography
- **Body Font:** System sans-serif stack (Tailwind default)
- **Code/Terminal blocks:** Monospace (`font-mono`)
- **Headings:** Bold + gradient clip (`bg-clip-text`)

### Key Components
- `.glass-card` вҖ” Glassmorphism panel with `backdrop-filter: blur(12px)`
- `.glow-avatar` вҖ” Profile image with dual-color radial glow
- `.scroll-dot` / `.scroll-dot.active` вҖ” Right-side section dots
- `.cursor` вҖ” Blinking typewriter cursor
- `.loading-animation` вҖ” Full-screen preloader overlay
